## Theme

Build something practical that solves a real problem YOU have.

Key principle: **practical impact beats fancy tech**. "This saved my roommate 20 minutes" is a stronger pitch than "this leverages ZK."

Prompt examples from organizers:
- What confuses you about DeFi? Could an app teach DeFi by making you actually use it?
- What's tedious in your crypto workflow? Is there a repetitive task you could automate away?

## Submission requirements

- **Name**
- **Description:** a few words on what the project is
- **Problem:** what problem you're solving
- **Solution:** how the project solves it
- **Project URL:** hosted web app URL (mobile: App Store link or Android APK link)
- **Github repo:** public repo URL
- **Category:** Monad Mainnet or Testnet (if both, pick Mainnet)
- **Contract address:** address deployed on Monad mainnet or testnet
- **Demo video:** max 3 min, uploaded to any video host, publicly visible URL
- **Post URL:** social media post URL (required for "Most viral solution" prize)

## What NOT to submit

- **AI slop:** AI use is fine, but it shouldn't be obvious the moment judges open the app. Fit the UI in the viewport, give the app a unique identity. The judging agent detects slop.
- **The tutorial special:** no todo apps, weather dashboards, or anything findable via "beginner React project ideas."
- **The mystery box:** no README, no demo video, no setup instructions, one commit titled "final final v2" = unscoreable. Judges must be able to run and understand it in 3 minutes.
- **Vaporware demos:** no fake buttons, no hardcoded success toasts. Judges always click it twice. One real feature beats five fake ones.
